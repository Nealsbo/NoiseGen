# NoiseGen
NoiseGen - Simple app for generating noise textures using Qt6 and numpy

---

# Features
Noise and image generation
- Noise generator:
    - White noise
    - Worley noise
- Image generator:
    - Color
    - Checker board

Post generator image modifier
- Modifiers for generated images:
    - Brightness

# Usage
```
python main.py
```

# Dependecies
At this point project uses numpy, Qt(PyQt6) and PIL(pillow)
